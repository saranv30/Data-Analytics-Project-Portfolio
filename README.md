## Project Overview

This repository contains multiple end-to-end data analytics projects across different domains including HR, Retail, E-commerce, and Telecom. Each project focuses on extracting insights and building predictive models to support business decision-making.

---

## Projects

### 1. HR Attrition Analysis

* Performed Exploratory Data Analysis (EDA) on employee data
* Built classification models to predict attrition
* Created Power BI dashboard for visualization
* Suggested strategies to reduce employee turnover

**Key Insight:** Employees with low salary and high overtime are more likely to leave.

---

### 2. Retail Performance Analysis

* Used SQL for data cleaning and querying
* Analyzed profit margins by category and region
* Studied relationship between inventory and profitability
* Built Tableau dashboard

**Key Insight:** Some high-selling products generate low profit due to poor margins.

---

### 3. E-commerce Return Analysis

* Analyzed return rates across categories and suppliers
* Built logistic regression model to predict returns
* Created Power BI dashboard for return risk

**Key Insight:** Specific product categories and suppliers contribute to higher returns.

---

### 4. Customer Churn Analysis

* Aggregated telecom data using SQL
* Built churn prediction model
* Segmented customers into risk categories

**Key Insight:** Customers with low engagement and frequent complaints are more likely to churn.

---

### 5. Customer Lifetime Value (LTV) Prediction

* Performed feature engineering (Recency, Frequency, Monetary)
* Built regression model to predict customer value
* Segmented customers based on LTV

**Key Insight:** A small group of customers generates most of the revenue.
